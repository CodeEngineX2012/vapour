A command line built for Linux/Unix based systems (basically systems that run Kernel code).
It comes pre-installed in the Skyzen Labs DevUSB

**How to run**

*Clone the github repository:*
- git clone https://github.com/CodeEngineX2012/vapour.git

*Clone the directory:*
- cd vapour/vapour/linux

*Make the .sh | Shell file executable (allow permisions):*
- chmod +x vapour.sh
- ./vapour.sh

*You will get:*
- usr@vapour ~ $
